 Instructions

 You will need the latest version of RStudio. 
 Be sure to have also installed R Tools and the Base R for Windows from CRAN.


 For this project, I used RStudio version 2022.07.1+554 (Spotted Wakerobin); 
 R base version 4.2.1; and R Tools version 4.2.


 Place all files in one clean directory before running the Rmarkdown files in RStudio. 
 All the required libraries will be installed if not present in your current RStudio 
 install. This is part of the code running in the Rmd files.

 There are graphic (png) files included which are knitted by Rmarkdown to the final
 output document. These are figures from cited documents and the summary of panel 
 regression outputs consolidated using Excel.

 The web (html) files (paper and dashboard) can be run on the fly without RStudio.
